Abstract

Nuestra iniciativa se centra en la creación de una herramienta con varios aspectos que fusionan un sistema de captura de tokens con una potente botnet. El componente denominado "TokenGrabber" está diseñado para recopilar los datos de las máquinas infectadas y enviarlos en formato JSON a un servidor central, donde se dispone de un panel de control para visualizar la información de todas las "víctimas". Por otro lado, la botnet incorporada dentro del troyano está equipada con un virus capaz de realizar conexiones de "reverse Shell", otorgándonos un amplio abanico de posibilidades, desde ejecutar ataques DDoS hasta llevar a cabo el minado de criptomonedas. El camuflaje del troyano se efectuará dentro de un popular videojuego conocido como "Superfetch Dog". Esta elección no es aleatoria, pues este juego destaca por su atractivo comercial, su capacidad para entretener a los usuarios y su demanda de recursos moderada, lo cual lo convierte en una opción altamente atractiva para nuestro propósito.

Introducción

La infraestructura de nuestro proyecto consiste en un servidor y varias máquinas virtuales dentro para ello vamos a utilizar el sistema de virtualización “VirtualBox”. Una de las maquinas llevara la “beef” tool (apartado beef tool), con la web y allí el troyano (apartado troyano). Otra tendrá la bbdd(apartado bbdd), en la otra habrá el visor de la bbdd (apartado visor) y el botmaster(apartado botmaster).
